Detailed Description

Author
silhavyj A17B0362P

The purpose of this class is to handle the logic of the game Connect4 - https://en.wikipedia.org/wiki/Connect_Four. This class is directly used by class Server when two players decide to play a game.

std::vector< std::pair< int, int > > Connect4::getWinningTiles ( std::vector< std::vector< std::pair< int, int >>> &  sequence)
private

Check is there is a winning sequence of four tiles in a row.

If there is a winning sequence, it will return the positions of the winning tiles. For example, [5,0];[4,0];[3,0];[2,0]. Otherwise, an empty vector will be returned.

Parameters
sequencein which we are looking for winning tiles (four in a row)
Returns
either an empty vector or a vector containing the positions of winning tiles
