Detailed Description

Author
silhavyj A17B0362P

The purpose of this class is to log every activity happening on the server so it could be used for analysis later on. Example of what is logged could be: when a client gets connected/disconnected, when they send a message, when they start playing game, when they not following the protocol, and so on. This class also stores everything the is printed out into a file named with the datetime when the server was run. This class is written as a singleton.

info message (for example, when a client gets connected to the server)

countdown message (for example, when waiting for a client to enter their nick)

booting message used when the server boots up

WARNING

GAME 

game message (when two clients play a game)

MSG
