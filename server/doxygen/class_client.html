Detailed Description

Author
silhavyj A17B0362P

This class holds information about a client that gets connected to the server. For example, it holds their nick, socket, state, etc.

Member Enumeration Documentation

◆ State

enum Client::State

State of the client.

Enumerator
NICK 

the client is supposed to enter their nick

LOBBY 

the client is in the lobby waiting for a game request, or they can send off one

SENT_RQ 

the client sent a game request and is waiting for a response from their opponent

RECV_RQ 

the client received a game request and is supposed to reply to it (accept/reject)

GAME 

the client is now playing a game

KILL_THREAD 

temporary state used for killing all threads associated with the client
